MovieNest

A full-stack **Movie Ticket Booking System** developed using **Python, Django, HTML, CSS, JavaScript, and SQLite**. 
This web application allows users to browse movies, create an account, log in, select seats, book tickets, and manage their bookings through a user-friendly dashboard.

Features
User Registration & Login
Browse Available Movies
Search Movies
Movie Details Page
Seat Selection System
Ticket Booking
User Dashboard
Booking History
Movie Reviews & Ratings
Responsive User Interface
Admin Panel for Managing Movies and Bookings
